# Descripcion breve del proyecto
Desarrollar un sitio web dinamico utilzando PHP, donde demostramos las aplicaciones practicas sobre variables, estructuras de control, arreglos y principalmente sesiones. El objetivo de este proyecto es colaborar equitativamente en equipo utilizando plataformas digitales como GitHub e InfinityFree para subir el servidor que trabajamos.

# Temas PHP implementados
Trabajamos principalmente en temas que ya habiamos visto en clase y que son muy importantes para poder usar PHP facilmente, entre ellas son:
- Variables
- Condicionales
- Bucles
- Arreglos
- Sesiones
